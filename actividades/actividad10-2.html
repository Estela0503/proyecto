La etiqueta video: para especificar la ubicación del archivo que se reproducirá, puede usar el elemento ≪source> dentro del elemento <video>. (Esto debe reemplazar el atributo src en la etiqueta <video> de apertura.) También puede utilizar varios elementos <source> para especificar que el vídeo está disponible en diferentes formatos. (Debido a un error en el iPad, debe proprcionar el vídeo MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: Este atributo especifica la ruta de acceso al vídeo.

type: Se debe utilizar este atributo para decirle al navegador qué formato es el vídeo. De lo contrario, descargará parte del vídeo para ver si se puede reproducir el archivo (que tomará tiempo y ancho de banda).

codecs: El códec que se usó para codificar el vídeo se suministra dentro del atributo type. tenga en cuenta el uso de comillas simples, así como comillas dobles en el atributo tipo, cuando se suministra
