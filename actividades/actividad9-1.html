ID Attribure

Cada elemento HTML puede llevar el atributo id. Se utliza para identificar de forma única ese elemento de otros elementos de la página. Su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carácter). Es importante que no hay dos elementos en la misma página tiene el mismo valor para sus atributos ID (de lo contrario, el valor ya no es único)

Dar un elemento una identidad única le permite darle un estilo diferente que cualquier otra instancia del mismo elemento de la página.

Por ejempo, es posible que desee asignar un párrafo dentro de la página (quizás un párrafo que contiene una cotización de extracción) un estilo diferente que todos los demas párrafos. En el ejemplo, el párrafo con el atributo ID cuyo valor es pullquote se hace en mayúsculas usando css.
